# R-news
Приложение с персонализированными новостями с регистрацией, анимациями и алгоритмами подбора потенциально подходящих пользователю новостей

В этом проекте я использовала:
* MVVM + binding
* SnapKit для верстки 
* Lottie для анимаций 
* UserDefaults для хранения данных
